# 信息工程系网站 · 原始需求文档 v1.1

> 状态：草稿 | 技术栈：Next.js 静态导出 | 数据：静态 JSON 文件

---

## 一、项目概述

**项目名称**：信息工程系官方网站

**核心定位**：院系信息展示平台 + 内容管理系统（CMS）

**用户角色**：
| 角色 | 说明 |
|------|------|
| 访客 | 无需登录，浏览所有前台页面 |

---

## 二、数据模型（静态数据 TypeScript 接口）

### 2.1 媒体 · Media
- 字段：文件名、文件类型、文件大小、alt 文本、url（相对 `/public`）
- 说明：所有图片和附件存放于 `/public/media/`，通过静态路径访问

### 2.2 新闻公告 · News
- 字段：标题、封面图、正文（富文本）、发布时间、状态（草稿/已发布）、分类标签
- 关联：无

### 2.3 专业 · Major
- 字段：
  - 基本信息：专业名称、所属院系、发布时间、责任编辑
  - 专业介绍：富文本
  - 人才培养：培养目标（富文本）、能力要求（富文本）、职业认证列表
  - 课程体系：主干课程列表（课程名 + 简介）
  - 技能培养：技能方向、岗位能力、实践能力（均富文本）
  - 就业方向：就业岗位列表、就业单位列表、方向说明
  - 图片集：关联 Media（多张）
  - 附件：附件列表（文件名 + 关联 Media）
- 关联：教师（多对多）、校企合作（多对多）、荣誉成果（一对多）

### 2.4 教师 · Teacher
- 字段：姓名、头像（关联 Media）、职称、学历、教学方向（富文本）、成果展示（富文本）、联系方式（可选）、排序权重
- 关联：专业（多对多）

### 2.5 荣誉成果 · Achievement
- 字段：标题、类型（竞赛/认证/科研/荣誉）、描述（富文本）、图片（关联 Media）、所属专业、时间
- 关联：专业

### 2.6 科研项目 · Research
- 字段：项目名称、类型（科研成果/项目/论文/专利）、描述（富文本）、发布时间、附件（关联 Media）

### 2.7 实训基地 · TrainingBase
- 字段：基地名称、类型（校内/校外）、介绍（富文本）、图片集（关联 Media）、合作说明

### 2.8 合作企业 · Partner
- 字段：企业名称、Logo（关联 Media）、合作模式（富文本）、合作项目列表
- 关联：专业（多对多）

### 2.9 招生信息 · Enrollment
- 字段：标题、类型（政策/计划/简章）、内容（富文本）、发布时间、附件（关联 Media）

### 2.10 就业数据 · Employment
- 字段：年份、就业率、就业分析（富文本）、就业单位列表

### 2.11 校友 · Alumni
- 字段：姓名、头像（关联 Media）、毕业年份、专业、简介、分类标签

### 2.12 校友活动 · AlumniActivity
- 字段：活动名称、时间、内容（富文本）、图片集（关联 Media）

### 2.13 轮播图 · Banner
- 字段：图片（关联 Media）、标题、跳转链接、排序、状态

### 2.14 院系概况 · DepartmentInfo（单例，对应 `/data/departmentInfo.ts`）
- 字段：系部介绍（HTML 字符串）、发展历程（时间线，数组）、办学特色（列表）、荣誉成果列表

## 三、页面需求

### 3.1 首页 `/`
| 模块 | 需求描述 |
|------|---------|
| 轮播图 | 自动 + 手动切换，点击跳转，数据来源 Banner collection |
| 新闻公告 | 展示最新 5 条，标题 + 时间，点击进详情，"更多"跳新闻列表 |
| 专业推荐 | 卡片展示 3-6 个专业（后台可配置推荐），名称 + 简介 + 跳转 |
| 快捷入口 | 4 个固定入口：招生咨询、就业信息、校企合作、外链（可配） |
| 活动图片 | 图文卡片，点击放大或跳详情 |

### 3.2 本系概况 `/about`
| 模块 | 需求描述 |
|------|---------|
| 系部介绍 | 富文本展示，数据来源 DepartmentInfo Global |
| 发展历程 | 时间线组件，按年份正序排列 |
| 办学特色 | 列表 + 图文结合展示 |
| 荣誉成果 | 图片 + 文字卡片列表 |

### 3.3 专业设置 `/majors`
| 模块 | 需求描述 |
|------|---------|
| 专业列表页 | 卡片 / 列表视图切换，展示全部专业，点击进详情 |
| 专业详情页 `/majors/[slug]` | 分 Tab 或分区展示：基本信息、专业介绍、人才培养、课程体系、技能培养、师资队伍、成果展示、校企合作、就业方向、图片、附件下载 |

### 3.4 师资队伍 `/teachers`
| 模块 | 需求描述 |
|------|---------|
| 教师列表 | 头像 + 姓名 + 职称卡片，可按职称筛选 |
| 教师详情 `/teachers/[id]` | 基本信息、教学方向、成果展示、联系方式（可选显示） |

### 3.5 教研科研 `/research`
| 模块 | 需求描述 |
|------|---------|
| 科研成果 | 列表 + 详情，支持类型筛选（成果/项目/论文/专利） |

### 3.6 实训基地 `/training`
| 模块 | 需求描述 |
|------|---------|
| 基地介绍 | 富文本 + 图片 |
| 实训环境 | 图片展示 + 说明 |
| 校内外基地 | 列表 + 详情，区分类型 |

### 3.7 校企合作 `/partners`
| 模块 | 需求描述 |
|------|---------|
| 合作企业 | Logo 墙 + 企业详情 |
| 合作项目 | 列表 + 详情 |
| 合作模式 | 富文本 + 图文展示 |

### 3.8 招生就业 `/enrollment`
| 模块 | 需求描述 |
|------|---------|
| 招生信息 | 分类展示（政策/计划/简章），支持附件下载 |
| 就业数据 | 就业率 + 数据可视化（可选）+ 分析文本 |
| 就业单位 | 企业列表，可分类筛选 |

### 3.9 校友会 `/alumni`
| 模块 | 需求描述 |
|------|---------|
| 校友信息 | 列表 + 详情，可按专业/届次分类 |
| 校友活动 | 列表 + 详情 + 图片 |

## 四、通用功能需求

| 功能 | 说明 |
|------|------|
| 富文本渲染 | 内容以 HTML 字符串存储在静态数据文件中，前台用 `dangerouslySetInnerHTML` 渲染（内容由团队维护，无 XSS 风险） |
| 图片展示 | Next.js Image 组件，支持点击放大（Lightbox） |
| 附件下载 | 文件放于 `/public/attachments/`，静态路径直接下载，显示文件名 + 类型图标 |
| 分页 | 列表页支持分页，默认每页 12 条 |
| 站内跳转 | Next.js Link，不刷页 |
| 外链跳转 | `target="_blank"` + `rel="noopener"` |
| SEO | 每个页面支持独立 meta title / description（Payload Global 或 Collection 字段） |
| 响应式 | 移动端适配（导航折叠、卡片单列） |

## 五、非功能性需求

| 项目 | 要求 |
|------|------|
| 部署环境 | 学校内网 Linux 服务器，Nginx 直接托管 `out/` 静态目录 |
| 构建方式 | `next build`（`output: 'export'`），产物为纯静态文件 |
| 文件存储 | 图片和附件放 `/public/`，随构建产物一起部署 |
| 首屏性能 | 全站 SSG，LCP < 2.5s（内网） |
| 浏览器兼容 | Chrome / Edge 最新两个大版本 |
| 内容更新 | 修改 `/data/` 目录下对应 TS 文件后重新构建部署 |

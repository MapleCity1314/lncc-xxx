# 课程群建设与数字化教学

课程建设以项目为驱动，按照“基础-应用-竞赛”的三级路径展开，配套数字化资源与标准化能力指标体系，确保每门课程的成果可度量。

## 课程梯度

- **基础阶段**：数据结构、计算机组成、工程制图等课程铺设基础能力。
- **应用阶段**：集成项目（桥梁施工模拟、路面养护计划）与微课程组件实现能力点打通。
- **提升阶段**：产教融合课题与竞赛训练，如“5G+智能运维”挑战赛。

## 数字化支撑

| 资源类型 | 应用方式 |
| ---------- | -------- |
| 数字教材平台 | 课程同步推送、多语言字幕、练习自动评分 |
| 实训虚拟机 | 一键复现真实工况，支持多人协同操作 |
| AI助教 | 通过对话接口解答细节问题、推送学习报告 |

## 成果检阅

课程建设委员会每学期以项目、考核与企业反馈三条维度评估，确保专业群建设与行业标准紧密对接。
